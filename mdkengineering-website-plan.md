## Brand Identity

### Core Positioning
**"Engineering intelligence into reality."**

MDK Engineering is not another AI consultancy. It's an engineering studio run by an inventor ‚Äî someone who bridges the gap between intelligent software and physical systems, from concept to working prototype to production. The brand should feel like walking into a high-end workshop where precision meets creativity.

### Brand Personality
- **Inventive** ‚Äî not corporate, not startup-bro, but genuinely curious and creative
- **Technical depth** ‚Äî the kind of person who can explain a transformer architecture AND solder a circuit board
- **German engineering meets academic rigor** ‚Äî TUM precision + Oxford research credibility
- **Approachable expertise** ‚Äî serious work, human delivery

### Visual Identity Direction
- **Aesthetic**: Industrial-refined. Think: dark workshop meets clean engineering drawings. Not the typical blue-gradient AI consultant look.
- **Color palette**: Dark charcoal/near-black base (#1a1a1a), warm white text (#f5f0eb), an accent color like engineering orange (#e8641b) or electric blue (#2d7ff9) ‚Äî something that pops against the dark like a status LED on a machine.
- **Typography**: A distinctive monospace or engineering-style display font (e.g., JetBrains Mono, IBM Plex Mono, or Space Mono) for headings. A clean sans-serif for body (e.g., DM Sans, Satoshi). This signals "I write code AND build things."
- **Visual motifs**: Blueprint-style lines, subtle grid patterns, photographs of real hardware and prototypes (not stock AI imagery), code snippets, CAD renders.

## Technical Recommendations

### Platform
**Recommended: Next.js or Astro deployed on Vercel**
- Fast, modern, great SEO
- You can code it yourself (which is on-brand for an engineer)
- Easy to add a blog with MDX or a headless CMS

**Alternative if speed matters more: Framer or Webflow**
- Faster to launch, still looks custom
- Good enough for V1 while you build the "real" site

### Performance & SEO Essentials
- Mobile-first responsive design
- Page load under 2 seconds
- Proper meta tags, Open Graph tags for social sharing
- Schema markup for local business (Munich)
- Google Search Console + Analytics from day one
- Alt text on all images

### Domain & Email
- Domain: mdkengineering.com (register via Cloudflare or Namecheap)
- Email: hello@mdkengineering.com (use Google Workspace or Zoho Mail)
- Set up SPF, DKIM, DMARC for email deliverability
